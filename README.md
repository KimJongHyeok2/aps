<pre>
<span>Window10(개발) / Ubuntu 16.04 LTS(배포)</span>
<span>Eclipse IDE</span>
<span>Tomcat 9.0.22</span>
<span>HTML/CSS3, JavaScript, JQuery</span>
<span>JSP</span>
<span>Bootstrap4</span>
<span>JDK 1.8</span>
<span>Spring 5.1.6</span>
<span>MyBatis</span>
<span>MySQL 5.7.27</span>
</pre>
